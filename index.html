Simple Social Counter

Best way to display share social count!

Supporting social site

Plugin support Linkedin, Facebook and Twitter

How to use


  • Download plugin
  • Add jquery.ssc.js inside head section after jQuery include
  • Enable Plugin with your parameters
Sample CSS styles
.ssc_main_twitter,.ssc_main_facebook, .ssc_main_linkedin{
 padding: 8px;
 float: left;}
.ssc_count_twitter,.ssc_count_facebook, .ssc_count_linkedin{
position: relative;
bottom: -3px;
font-size: large;
font-weight: bold;
color: #333;
margin-left: 3px;

Examples

Single call
$(document).ready(function ()
            {
                $('#id_your_element').ssc();
            
            });
Single call with custom additional parameters
$('#id_your_element').ssc({
url:'test1.html',
social:'linkedin,facebook,twitter' ,
facebook_ico:'your_custom_ico_url',
linkedin_ico:'your_custom_ico_url',
twitter_ico:'your_custom_ico_url'
});
Multiple call in loop
$('.element').each(function ()
                {
                    var address = $(this).find('a').attr("href");
            
                    $(this).find('.results_loop').ssc({ url: address,social:'linkedin,twitter' });
                });
